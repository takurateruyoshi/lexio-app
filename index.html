プレイ履歴 / AIの読み

レキシオのルール

牌の強さ

3 が最弱、2 が最強: 3 < 4 < … < 最大数字 < 1 < 2
同じ数字はスートで決着: ☁雲 < ★星 < ☾月 < ☀太陽

役（同じ枚数同士でのみ勝負）

単騎（1枚）／ ペア（同数字2枚）／ トリプル（同数字3枚）／ 5枚役

5枚役の強さ（弱 → 強）

ストレート < フラッシュ < フルハウス < フォーカード < ストレートフラッシュ

ストレートは連番5つ。上端をまたぐ形（例: 6-7-8-9-1、7-8-9-1-2）も有効。 2 を超えて 3 へは続きません。

進行

リーダーが好きな役を出し、以降は同じ枚数でより強い役を出すか「パス」。 全員がパスすると場が流れ、最後に出した人が次のリーダーになります。

ラウンド終了と精算

誰かが手札を出し切った瞬間にラウンド終了。 全プレイヤーが互いに「残り枚数の差」をチップで支払います。 手札に残った 2 一枚ごとに自分の支払い倍率が +1（2倍、3倍…）。

設定したラウンド数を繰り返し、累計チップで総合順位が決まります。

結果

チップは「残り枚数の差」を支払い、手札の2一枚ごとに 支払い倍率が +1（2倍, 3倍…）になります。
